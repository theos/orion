An enumeration indicating that the property should use the assign attribute.

  • assign

    Indicates that the target object is not responsible for keeping the property alive.

    This attribute cannot be combined with any other attributes.

    Warning

    If the underlying value is deallocated, it becomes a dangling pointer and accessing it is undefined behavior.
